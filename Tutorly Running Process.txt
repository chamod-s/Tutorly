Tutorly Running Process (C# WPF & MySQL Native Desktop Application)

This application is a fully native desktop application built with C# and WPF (.NET 10.0), connecting directly to a local MySQL database managed via MySQL Workbench.

Prerequisites
Before starting, ensure you have:
1. .NET 10.0 SDK installed on your computer.
2. MySQL Server and MySQL Workbench installed and running.

Step 1: Set Up the Database in MySQL Workbench
1. Open MySQL Workbench.
2. Connect to your local MySQL instance.
3. Open a new Query Tab and run:
   CREATE DATABASE tutorly;
4. Set "tutorly" as your active database schema.
5. Open and execute the schema.sql file located in the project root:
   - File -> Open SQL Script... -> select "schema.sql" -> Run all statements (lightning bolt icon).
6. Open and execute the seed.sql file located in the project root to insert test data:
   - File -> Open SQL Script... -> select "seed.sql" -> Run all statements.

Step 2: Configure Database Credentials in C# (Optional)
If your local MySQL username is NOT "root" or your password is NOT "CJS@12345":
1. Open "desktop_native/Database/DatabaseHelper.cs" in any text editor.
2. Update the connection string with your credentials:
   private static readonly string ConnectionString = "Server=localhost;Port=3306;Database=tutorly;Uid=YOUR_USER;Pwd=YOUR_PASSWORD;AllowUserVariables=True;UseAffectedRows=True";

Step 3: Run the Application
1. Open a terminal (PowerShell or Command Prompt) and navigate to the project directory:
   cd desktop_native
2. Run the application using the .NET CLI:
   dotnet run
